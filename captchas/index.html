Hello there and welcome to the super-simple captcha card tool!
Sadly, until I have the time to recreate the graphics generator, there are two versions of this site.

Version 1 includes only the calculator.
Version 2 includes the calculator and graphics generator.

If you are using chrome use version 1 or switch to a supported browser.

If you are using Internet Explorer or Microsoft Edge, use version 2.

If you are using Firefox, ensure you have the flash plugin installed. (You can get it here.) Then use version 2.

If you are using some other browser, try version 2 first, and if it doesn't work, attempt version 1, or switch browsers.
